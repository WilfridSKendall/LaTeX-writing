# LaTeX-writing
Useful packages for writing mathematical documents in LaTeX.

## WSKmaths --- useful mathematical definitions.

This package loads useful mathematics packages, then provides a number of useful environments, and defines a number of significant macros to facilitate writing of mathematics.


## WSKtracking --- commands related to tracking changes.

This package loads useful mathematics packages, then provides environments and macros to facilitate revision annotations.


## Further details

See the PDF files in WSKtracking and WSKmaths directories.

See also LaTeX-writing.tex (and LaTeX-writing.pdf) for a basic example.
